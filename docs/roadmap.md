# Roadmap - Vault Bank API

## MVP
- CRUD Customer e Account
- Depósito, saque e transferências
- Extrato por conta

## Segurança
- JWT, roles (USER/ADMIN)
- Hash de senha com BCrypt

## Confiabilidade
- Transações atômicas
- Pessimistic e Optimistic Locking
- Idempotência

## Observabilidade
- Logs estruturados
- Métricas e Health endpoints
- Testes de integração e concorrência

## Futuro
- Event Sourcing e CQRS
- Integração com gateway de pagamentos
- Dashboard de analytics
